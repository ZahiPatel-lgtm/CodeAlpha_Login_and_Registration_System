## Login and Registration System

This project is a simple Login and Registration System developed in C++.  
The program allows users to register using a username and password and stores the credentials in a file. Users can then log in by entering the correct credentials, which are verified from the stored file data.

### Features
- User Registration
- User Login Authentication
- Stores user credentials in a file
- Displays success and error messages
- Simple and beginner-friendly implementation

### Concepts Used
- File Handling (`ifstream`, `ofstream`)
- Strings
- Conditional Statements
- Loops
- User Input and Output
